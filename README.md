# Sangha CODE Test Reproduction Files

This repository contains the files used to reproduce the Sangha-style CODE test comparison for the ECG image classification manuscript.

## Contents

- `data/images_sangha_style_rendering/`: CODE test ECG images rendered in the Sangha-style image format.
- `data/gold_standard.csv`: reference labels for the evaluated CODE test subset.
- `data/prediction_sanghaweb_s_20250804_014051.csv`: predictions exported from the Sangha web model on the same rendered images.

## Notes

The files are provided to support reproducibility of the literature-comparison analysis. The ECG images are derived from the CODE test subset and rendered as images for model comparison.

The Sangha web-model predictions were obtained from <https://www.cards-lab.org/ecgdx>.
